# 0x12. JavaScript - Warm up

## Learning Objectives

### 1. Why JavaScript programming is amazing
JavaScript is a versatile and powerful programming language that runs in the browser, enabling dynamic and interactive web development. Explore the amazing capabilities of JavaScript and understand its significance in modern web development.

### 2. How to run a JavaScript script
Learn the various ways to execute JavaScript code, whether it's in a browser console, through an HTML file, or using server-side environments.

### 3. How to create variables and constants
Discover how to declare variables and constants, and understand their differences in terms of scope, reassignment, and immutability.

### 4. Differences between var, const, and let
Explore the nuances between `var`, `const`, and `let` in JavaScript and how they impact variable declaration, scope, and hoisting.

### 5. Data types available in JavaScript
Understand the different data types in JavaScript, including primitive types (e.g., string, number, boolean) and complex types (e.g., object, array).

### 6. How to use if, if ... else statements
Master the basics of conditional statements in JavaScript to control the flow of your code.

### 7. How to use comments
Learn the importance of comments in your code and how to use them effectively for documentation and readability.

### 8. How to assign values to variables
Explore various ways to assign values to variables and understand the implications of these assignments.

### 9. How to use while and for loops
Dive into the world of loops – understand the syntax and usage of `while` and `for` loops for efficient code execution.

### 10. How to use break and continue statements
Learn how to control loop execution with `break` and `continue` statements for more precise control flow.

### 11. What is a function and how to use functions
Discover the concept of functions, how to define them, pass parameters, and execute reusable blocks of code.

### 12. What does a function that does not use any return statement return
Understand the behavior of functions that do not explicitly use a `return` statement and their implicit return values.

### 13. Scope of variables
Explore variable scope – learn about global and local scopes and how they affect variable accessibility.

### 14. Arithmetic operators and how to use them
Master arithmetic operators in JavaScript and understand their role in performing mathematical operations.

### 15. How to manipulate dictionaries
Learn how to work with JavaScript objects (similar to dictionaries in other languages) to store and manipulate key-value pairs.

### 16. How to import a file
Understand the basics of importing files in JavaScript, whether it's for modularization or integrating external code.

Feel free to explore these topics in order or jump around based on your needs. Happy coding! 🚀👩‍💻👨‍💻
